# WorkingPlace
## Gestionnaire de virement bancaire maison 100 % maison 
L'objectif de ce gestionnaire est de pouvoir via un menu : 

- Ajouter une tranaction banacaire 
- Modifier une transaction bancaire 
- Voir ces transaction bancaire et les Trier par date 
- Les statistiques de virement par pays 
- Exporter toutes ces transactions bancaire via un fichier .json 
-Importer un fichier json est analyser le contenue via : 
    - Filtre du pays 
    - Filtre de compte
